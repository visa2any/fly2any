### WEEKLY OPERATIONAL WORKFLOW

**Monday: Query Performance Review**
1. Pull top 1,000 queries from GSC API
2. Flag any with CTR >20% below expected for position
3. Identify queries with dwell time <30 seconds
4. Generate Query Health Score (QHS) for each

**Tuesday: Content-Query Alignment**
1. Map underperforming queries to specific pages
2. Analyze content for intent mismatch
3. Assign fix priority (P0-P2)

**Wednesday: Fix Implementation**
1. P0: Rewrite content for intent alignment (same day)
2. P1: Add missing information (48 hours)
3. P2: Structural improvements (1 week)

**Thursday: User Session Analysis**
1. Sample 100 sessions for pogo-sticking queries
2. Record screen sessions (with consent) for UX issues
3. Identify drop-off points in conversion funnels

**Friday: System Calibration**
1. Adjust CTR expectations based on SERP features
2. Update intent classification model
3. Deploy A/B tests for top 10 underperforming queries
